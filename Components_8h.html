Detailed Description

Defines core gameplay component structures for ECS entities.

This file contains a set of commonly used component types, including physics, rendering, and gameplay-related data. Intended for rapid prototyping and small-scale projects, all components are kept in one place for simplicity.

As the component set expands, it may be beneficial to refactor into module-specific files.
