Detailed Description

Represents a lightweight handle used to identify components in the ECS.

Internally, this wraps pair of values (id + version) to ensure uniqueness. Also includes hash similar to boost's hash-combine implementation: https://www.boost.org/doc/libs/1_55_0/doc/html/hash/reference.html#boost.hash_combine
