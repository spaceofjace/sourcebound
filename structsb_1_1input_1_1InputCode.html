Detailed Description

Represents a single user input binding.

Combines a device type (keyboard, mouse, or gamepad), an input kind (button vs. axis), and a variant payload (InputData) that stores the appropriate device-specific code.

DeviceType and InputKind will be set to Invalid if not initialized/parsed correctly.
